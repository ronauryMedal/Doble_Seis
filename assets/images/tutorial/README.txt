Capturas para la guía "Cómo usar la app".

Coloca aquí capturas reales de la app con EXACTAMENTE estos nombres para que
reemplacen automáticamente las ilustraciones de respaldo en la guía:

Antes de empezar:
  guide_mode.png        -> selector de modo (Equipo vs Equipo / Individual)
  guide_players.png     -> cantidad de jugadores y nombres
  guide_target.png      -> meta de puntos (100 / 150 / 200 / manual)
  guide_connection.png  -> compartir marcador (Solo local / WiFi local / Nube)
  guide_spectator.png   -> tarjeta "Unirme como espectador"

Durante la partida:
  guide_toolbar.png     -> barra superior (reloj, bitácora, nueva partida)
  guide_select.png      -> tarjetas de equipos/jugadores seleccionando uno
  guide_score.png       -> teclado anotando (+30 y número personalizado)
  guide_events.png      -> eventos especiales (Capicúa / Tranque)
  guide_fix.png         -> bitácora corrigiendo/eliminando una anotación
  guide_share.png       -> QR para compartir en vivo por WiFi
  guide_end.png         -> celebración del ganador (revancha / cambiar jugadores)

Después de jugar:
  guide_stats.png       -> historial / estadísticas

Sugerencia: usa imágenes verticales (ej. 1080x1920 o recortadas) en PNG.
Si un archivo no existe, la guía muestra una ilustración dibujada en su lugar.
